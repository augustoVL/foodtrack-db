# FoodTrack DB

FoodTrack is a database project designed to manage the operations of food trucks across different locations in a city.

The system stores and organizes information about:

* Foodtrucks
* Products
* Locations
* Orders
* Order item details

The project is built using **Microsoft SQL Server** and managed through **DBeaver** as the database client.

All database schema versions are controlled using **Git and GitHub**, simulating a professional development workflow.
